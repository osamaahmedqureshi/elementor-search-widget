# Elementor Search Widget

A lightweight WordPress plugin that adds a custom Elementor search widget without requiring Elementor Pro.

The plugin includes multiple search layouts for Elementor pages, including Classic, Minimal, Creative, Full Screen, and Half Screen modes.

## Features

- Custom Elementor search widget
- Works with Elementor Free
- Classic, Minimal, Creative, Full Screen, and Half Screen layouts
- AJAX search suggestions
- Search posts, pages, and WooCommerce products when WooCommerce is active
- Customizable input, button, overlay, and panel styles from Elementor controls
- Lightweight CSS and JavaScript assets

## Requirements

- WordPress 5.8 or higher
- PHP 7.4 or higher
- Elementor installed and active
- WooCommerce is optional and only required for product search

## Installation

1. Download the plugin ZIP from the GitHub Releases section.
2. Go to **WordPress Dashboard > Plugins > Add New > Upload Plugin**.
3. Upload the ZIP file.
4. Activate the plugin.
5. Edit a page with Elementor.
6. Search for **Search** under the Elementor widgets panel.

## Usage

After activation, open any Elementor page and add the Search widget. Choose the layout, search source, placeholder text, button type, and styling options from the widget controls.
